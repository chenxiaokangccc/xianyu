# 闲鱼爆款监控

内部使用的闲鱼商品链接监控原型。当前版本是可直接打开的静态网站，数据保存在浏览器本地，支持上传商品链接、每日自动更新模拟数据、记录历史变化、查看商品详情和导出 CSV。

## 使用方式

最简单的方式：双击 `打开监控网站.command`。

也可以用本地采集服务打开，这样“立即更新”才能真实采集闲鱼页面：

```bash
python3 server.py
```

然后访问 `http://localhost:4173/index.html`。

## 已支持

- 上传多条闲鱼商品链接，支持粘贴文本或上传 txt/csv。
- 每天首次打开时自动更新一次，也可点击“立即更新”手动采集。
- 记录标题、账号名称、价格、想要人数、浏览量、内容文案、状态、备注。
- 表格展示今日新增想要、今日新增浏览、是否改价、是否改标题、是否改文案、正常/失败/下架状态。
- 商品详情展示当前数据、历史价格曲线、历史想要曲线、历史浏览曲线和变化标记。
- 导出当前筛选结果为 CSV。
- 商品列表和历史记录由服务端共享保存，两个人访问同一个网址会看到同一份数据。

## 真实采集说明

真实闲鱼采集已经放到 `server.py`。网页会请求 `/api/xianyu/snapshot`，本地服务会访问商品链接并尝试解析：

- 商品标题
- 账号名称
- 价格
- 想要人数
- 浏览量
- 内容文案

如果闲鱼要求登录、出现安全验证，或页面没有公开返回这些字段，商品会显示“失败”，备注会提示需要在“采集设置”里填写 Cookie 后重试。
